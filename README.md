# godot-project-base
A base for Godot projects.

## Features

- CheeGames intro
- `StateMachine` class
- `State` class
- `DamageManager` class
- `AwaitTimer` class
- `Globals` class
- `ShadowGenerator` class
- `SoundManager` class
- `Gamemanager` singleton
- `Settings` singleton
- `Pathinding` singleton
- `Cursor` singleton
- `LoadingZone` scene
- `PlayerDetector` scene
- `Entity` scene
- A topdown player controller
- Hurt and Hitboxes
- Screenshake
- Flash effect
- Camera mouse weight
